#flipkart project

A simple eCommerce website built using **HTML, CSS, JavaScript, and JSON**.

---

##  Project Files

###  HTML
- index.html → Home page  
- product.html → Product listing  
- product-details.html → Product details  
- cart.html → Cart page  
- buy.html → Buy now page  
- login.html → Login page  
- order-success.html → Order confirmation  

---

###  CSS
- styles.css → All styling (layout, navbar, cards, buttons)

---

###  JavaScript
- products.js → Display products  
- product-details.js → Product details & actions  
- cart.js → Cart functionality  
- buy.js → Buy now functionality  
- order-success.js → Order display  

---

###  JSON
- products.json → Product data  

---

##  Features

- View products by category  
- View product details  
- Select size (if available)  
- Add to cart  
- Buy now option  
- Increase / decrease quantity  
- Remove items  
- Checkout and order confirmation  

---

##  Storage

- cart → stores cart items  
- buy → stores buy now items  
- order → stores final order  

---
